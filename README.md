## Selected Projects

### [Nexus Express System](https://github.com/VoVanTuTai/logistics-management-system)

A logistics management platform for shipment creation, pickup, hub operations, courier delivery, public tracking, COD settlement, and operational reporting.

**Tech stack:** TypeScript, NestJS, React/Vite, PostgreSQL, Prisma, RabbitMQ, Docker, Redis, MinIO

Highlights:

- Designed a microservices-oriented system with Gateway/BFF, domain services, database-per-service ownership, and RabbitMQ domain events.
- Built and documented logistics workflows for shipment lifecycle, pickup, dispatch, manifest, scan, delivery, tracking, reporting, and payment.
- Created OpenAPI contracts and event payload examples for service communication.
- Implemented dispatch and scan modules including task assignment, current-location tracking, outbox/idempotency flows, and local Docker Compose infrastructure.

### [Tourist Accommodation Management System](https://github.com/VoVanTuTai/Tourist_Accommodation_Management_System_ThieuNu)

A booking and accommodation management system for customers, providers, and administrators.

**Tech stack:** Node.js, Express.js, EJS, MySQL, Sequelize, VNPay, QR code, PDF/Excel reporting

Highlights:

- Built booking flows, room/provider management, customer accounts, admin dashboards, and commission reports.
- Integrated VNPay payment routes and QR payment pages.
- Implemented PDF and Excel export for commission/reporting workflows.

### [E-commerce Microservices](https://github.com/VoVanTuTai/22655911-VoVanTuTai_EProject)

A small e-commerce microservices system with Auth, Product, and Order services behind an API Gateway.

**Tech stack:** Node.js, Express.js, MongoDB, RabbitMQ, Docker Compose, JWT, Mocha

Highlights:

- Built separate Auth, Product, and Order services with independent MongoDB databases.
- Used RabbitMQ pub/sub for asynchronous communication between Product and Order services.
- Added API Gateway routing, JWT authentication, Docker Compose setup, health checks, and Mocha tests.
